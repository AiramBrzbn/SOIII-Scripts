Practica 1
Listar los repositorios actualmente en uso
dnf repolist all
para ver solo los activos
dnf repolist enabled
Actualizar los repositorios del sistema:
sudo dnf update && upgrade

buscar la herramienta en el caché de los repositorios
dnf search bashtop

si esta disponible en el repositorio
sudo dnf install bashtop -y
si no lo instalamos de manera manual con:
cd /opt
sudo git clone https://github.com/aristocratos/bashtop.git
cd bashtop
sudo make install
si no contamos con la herramienta make install la descargamos asi:
sudo dnf groupinstall "Development Tools" -y

bashtop

Desinstalarla con dnf
sudo dnf remove bashtop -y
Si se instaló manualmente
sudo rm -rf /opt/bashtop
sudo rm /usr/local/bin/bashtop

rm -rf ~/.config/bashtop
sudo dnf autoremove -y
